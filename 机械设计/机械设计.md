	+ 推力轴承只能承受轴向力
		+ 推力球轴承，只承受轴向力载荷，型号代号5
	+ 角接触轴承既可以承受径向力也可以承受轴向力
		+ 主要代表：角接触球轴承，型号代号7
		+ 主要代表：圆锥滚子轴承，型号代号3
基本额定动载荷（寿命为10\*6的载荷）
基本额定寿命：按一组轴承中10%的轴承发生点蚀破坏而90%不发生点蚀破坏前的转数，
当量动载荷：

		+ 滑块联轴器：存在偏心，会造成动载荷

		+ 齿式联轴器

#### 轴
轴的类型：传动轴，转轴，心轴
传动轴：在工作中只承受扭矩不承受弯矩的轴
转轴：在工作中既承受弯矩又承受扭矩的轴
心轴：在工作中只承受弯矩不承受扭矩的轴
### 四道计算
#### 1、极限应力线图、疲劳强度
+ 单项稳定变应力时疲劳强度的计算
	 + 1、计算出危险截面上的最大工作应力$\sigma_{max}$和最小工作应力$\sigma_{min}$，然后计算出$\sigma_{m}$和$\sigma_{a}$，然后在极限应力线图上找到该点M
	 + 2、根据不同情况确定疲劳极限的点M`
		+ $r=C（常数）时$
	 		$可以推出：\frac{\sigma_a}{\sigma_{m}} = C$因此，极限应力点在过M点和坐标原点的连线上，即：坐标原点和M点的连线与零件应力线图的交点处，由此可以求得对应点M的疲劳极限
			然后校核最大工作应力，即$\frac{\sigma_{max}`}{\sigma_{max}}>= S(S为安全系数)$
		+ $\sigma_{m} = C时$
			可想而知，竖直一条线过M点，与极限应力线图相交的点就是对应的疲劳极限，然后校核最大工作应力，即$\frac{\sigma_{max}`}{\sigma_{max}}>= S(S为安全系数)$
		+ $\sigma_{min} = C时$
			可想而知，$\sigma_{min} = C = sigma_{m} - sigma_{a}$即：$\sigma_{a} = sigma_{m} - C$
			因此，应该是过M点的方程y = x-C与极限应力线图的交点为对应的疲劳极限，然后校核最大工作应力，即$\frac{\sigma_{max}`}{\sigma_{max}}>= S(S为安全系数)$
#### 2、紧螺栓连接受横向力，轴向力作用时的强度计算，同时承受径向力（难！）
**预备知识**
1、对于螺栓组有轴向力，横向力，扭矩，翻倒力矩四种情况，但是对于单个螺栓本身的受力只有两种情况，受剪和受拉，受剪即：用螺栓自身平衡外载荷；受拉即：用接触面的摩檫力平衡外载荷；
2、受剪螺栓情况下的强度校核条件为：$$\frac{4F_s}{{\pi}d^{2}m}<[\tau]$$
3、受拉螺栓分两种情况：
	一种是只受预紧力，例如受到横向外载荷选择使用受拉螺栓的时候，横向力被摩擦力平衡掉，螺栓只受预紧力。此时强度条件为$$\frac{1.3F^{`}}{{\pi}{d_1^2/4}}<[\sigma],F^{`}为预紧力$$
	另一种是既受预紧力又受工作载荷F，例如外载荷是轴向载荷F或者翻倒力矩M时，此时强度条件为$$\frac{1.3F_0}{{\pi}d_1^2/4}<[\sigma],F_0为螺栓总拉力，计算方式如下：$$
	$$F_{0} = F^{``} + F$$
	$$F_{0} = F^{`} + \frac{c1}{c1+c2}F$$
	式中$F^{``}为残存预紧力，\frac{c1}{c1+c2}为螺栓相对刚度，F为工作载荷$
4、如果受变载荷还需要验算螺栓的应力幅，即$\sigma_{a}<[\sigma_{a}]$,应力幅的强度公式为：$$\sigma_{a} = \frac{c1}{c1+c2} \frac{2F}{{\pi}d_1^2}<[\sigma_{a}]$$
**解题方法：**
+ 1、受力分析
	+ 将外力移动到螺栓组的几何中心，然后分散到各个螺栓上
	+ 将力偶力矩拆分转化为垂直于几何中心和螺栓中心的几个力
	+ 计算受合力最大的那个螺栓
+ 2、根据是什么形式的螺栓（受拉，受剪）来计算预紧力
+ 3、校核强度条件
#### 3、斜齿轮，圆锥齿轮的蜗杆传动的综合受力分析
**预备知识**
1、两个斜齿轮啮合，旋向要相反
 
**解题方法：**
+ 1、判断齿轮情况
	+ 判断齿轮的转动方向，传动关系（哪个是主动轮哪个是从动轮）
	+ 如果是斜齿轮，判断齿轮的旋向，轴线竖起来，螺旋线哪边高就是哪旋

+ 2、受力分析：斜齿轮的三个力：圆周力，径向力，轴向力
	+ 圆周力Ft与啮合点速度方向主反从同（主动轮圆周力与啮合点速度相反，从动轮与啮合点速度相同）
	+ 径向力Fr永远指向圆心（轴线）
	+ 轴向力Fa根据相应的左右手定则判断（仅适用于主动轮，左旋用左手右旋用右手）
+ 2（1）、受力分析：锥齿轮的三个力：圆周力，径向力，轴向力
	+ 圆周力Ft与斜齿轮相同
	+ 径向力Fr也与斜齿轮相同
	+ 轴向力Fa由小端指向大端
	+ 名义法向载荷Fn即实际受力
+ 3、计算各个力的大小：
	+ 下手点：圆周力Ft = 2T/d
	+ 其他力根据与圆周力Ft的关系计算
**蜗轮蜗杆的解题方法：**
+ 1、判断蜗轮蜗杆的情况

+ 2、受力分析

	+ 轴向力：蜗杆轴向力等于蜗轮圆周力
	+ 圆周力：蜗杆圆周力等于涡轮轴向力

#### 轴系改错题：（只需要指出错误，不需要改正）
四个方面：
+ 定位
	+ 轴向定位：不能左右移动，常用套筒，轴肩，端盖
	+ 周向定位：不能随意旋转，常用键
+ 装配
	+ 不能出现凹形结构，无法装配
	+ 端盖，透盖间隙密封，要有间隙（防止动静干涉），还要加毡圈密封（防止进入灰尘和润滑油泄露）
	+ 端部倒角，方便零件装入，导向作用
	+ 轴承前段有台阶
	+ 轴段短于齿轮，为了提供一个可靠的装配定位
+ 拆卸
	+ 定位低于轴承内圈，容易拆卸
+ 工艺
	+ 端部轴端越程槽
	+ 螺纹段退刀槽，防止车刀伤到边上轴段
	+ 键长短于轴段
	+ 如果有多个键，要位于同母线
