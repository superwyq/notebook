计时器计数器的Mode：

| Mode  | M1  | M0  | Operating Mode                          |
| ----- | --- | --- |:--------------------------------------- |
| Mode0 | 0   | 0   | 13位，很少用                            |
| Mode1 | 0   | 1   | 16位，初始化时要设置TH和TL              |
| Mode2 | 1   | 0   | 8位，初始化TH自动Copy到TL，工作时只用TL |
| Mode3 | 1   | 1   |                                         |

   TR
